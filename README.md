# 🔐 Login & Registration System (C++) 🔐

**Securely create accounts and log in with C++ 💻🪟**

---

## 📝 Project Description

The **Login & Registration System** is a C++ console program that allows users to **create accounts, log in, and manage credentials securely 🔒**.  
It uses **windows.h 🪟** to display **colored text** for clear feedback and a better user experience.  

The program demonstrates:  
- **User Input Validation ⌨️**  
- **Duplicate Username Checking ❌**  
- **Success ✅ and Error ❌ Messages**  
- Basic **File Handling 📁** for storing user data  

---

## ⚙ Features

- **Register new accounts** with unique usernames  
- **Login** with stored credentials  
- **Color-coded messages** for success, warnings, and errors  
- **Easy-to-read console interface** using windows.h  
- **Safe handling of user input**
